# BioMiLLM
BioMiLLM: A biology-driven, interpretable large language model for accurate and robust Type II antitoxin prediction under data scarcity and class imbalance.

The source code of BioMiLLM: Biological Mutation-Augmented Interpretable Large Language Model for Accurate and Robust Antitoxin Prediction will be released upon acceptance of the paper.
